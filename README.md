# CBNU_Blind_date_matching


- 프로젝트 설명 : 교내 소개팅 프로그램
  
  외적인 요소를 중시하는 보편적인 소개팅의 단편적인 측면을 배제하는 것이 주 목적으로,서로 마음이 맞는 사람들을 연결해주기 위해 비슷한 “관심사” 우선 매칭 소개팅 프로그램

- 개발기간 : 2023.10 ~ 2023.12 




## Features

- 회원가입 & QRcode 학교인증
  
- 관심사 선택을 기반으로 매칭기능
  
- 채팅기능
